| File              | Responsibility                                                                     |
| ----------------- | ---------------------------------------------------------------------------------- |
| **app.js**        | Starts the application. Calls initialization functions.                            |
| **navigation.js** | Builds the navigation from `navigation.json` and handles menu clicks.              |
| **renderer.js**   | Loads HTML views into the main content area.                                       |
| **router.js**     | Will manage URLs, browser back/forward buttons, and deep linking (next milestone). |
| **theme.js**      | Reserved for future theme handling (currently unused).                             |
| **utils.js**      | Generic helper functions shared across modules.                                    |
